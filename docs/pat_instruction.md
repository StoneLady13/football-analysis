# Génération d'un Personal Access Token (PAT)

## Procédure

1. Aller dans GitHub > Profile > Settings
2. Cliquer sur Developer settings
3. Personal access tokens
4. Generate new token (classic)
5. Donner un nom et une expiration
6. Sélectionner les scopes nécessaires
7. Générer le token

## Scopes sélectionnés

- repo : accès complet au dépôt
- workflow : gestion des workflows (si tests CI/CD)

⚠ Le token ne doit jamais être commité dans le dépôt.
Il est utilisé uniquement pour authentification locale.
